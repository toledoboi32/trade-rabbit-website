## 🎯 **Getting Started**

### **1. Quick Setup (15 minutes)**
```bash
# Clone or download the website
git clone https://github.com/traderabbit/website.git
cd traderabbit-website

# Install dependencies (optional)
npm install

# Start local server
npm start
# or
npx serve . -p 3000
```

### **2. Deploy to Production**
```bash
# Deploy to Vercel
npm run deploy

# Or upload to any web host
# Files are in the root directory
```

### **3. Customize Your Content**
1. Replace placeholder images in `/assets/images/`
2. Add your demo videos to `/assets/videos/`
3. Update contact information in `index.html`
4. Modify testimonials with real customer quotes

## 🎨 **Customization Guide**

### **Adding Your Content**

#### **1. Replace Demo Video**
```html
<!-- In index.html, update the video source -->
<video controls poster="assets/images/your-demo-thumbnail.jpg">
    <source src="assets/videos/your-demo-video.mp4" type="video/mp4">
</video>
```

#### **2. Update Contact Information**
```html
<!-- Update phone numbers throughout the site -->
<a href="tel:+1YOURNUMBER" class="demo-button">
    Call Demo Line: (YOUR) NUMBER
</a>
```

#### **3. Add Customer Testimonials**
```html
<!-- Add real customer testimonials -->
<div class="testimonial-card">
    <p class="testimonial-content">"Your customer's actual quote..."</p>
    <div class="testimonial-author">
        <img src="assets/images/customer-photo.jpg" alt="Customer Name">
        <div class="author-info">
            <h4>Customer Name</h4>
            <p>Company Name</p>
        </div>
    </div>
</div>
```

#### **4. Update Company Information**
```javascript
// In package.json, update your details
{
  "author": {
    "name": "Your Company Name",
    "email": "your-email@company.com",
    "url": "https://your-website.com"
  }
}
```
